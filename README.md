# country.sk
СЛИВ СКРИПТА для РП-стран на Skript


Сливаю очень жесткий скрипт на РП-Страны (месть одному проекту юзащий этот скрипт)
Гайд по установке:
- Устанавливаем Skript (https://github.com/SkriptLang/Skript)
- Заходим в папку Skript/scripts
- Закидываем туда файлик со скриптом [(жми)](https://github.com/meigoc/country.sk/releases/tag/0.5)
- В игре пишем /sk reload (название файла, то есть pizdec.sk)
- Юзаем скрипт
Если что-то не понятно, то можете обратиться в ЛС

#меню - страны
#страны - открыть меню с выбором президент или житель, если нет президента
#страны - стать гражданином, если есть
#4 часа инактива = снятие с президента
